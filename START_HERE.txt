⚙️ TECH STACK RATIONALE
═════════════════════════════════════════════════════════════════════════════════

FRONTEND
  Next.js 14      - Full-stack, API routes, SSR, optimal for startups
  React 18        - Modern UI library, hooks support
  TypeScript      - Type safety, better DX, catch errors early
  Tailwind CSS    - Rapid UI development, responsive by default

BACKEND
  Next.js Routes  - Serverless APIs, no separate backend needed
  Node.js         - JavaScript runtime, same language frontend to backend
  Prisma          - Type-safe ORM, migrations, excellent queries

DATABASE
  PostgreSQL      - ACID compliance, reliable, great for e-commerce
  Prisma Client   - Type-safe database access

REAL-TIME
  Socket.io       - Simple WebSocket, good documentation, scalable

EXTRA
  NextAuth.js     - Built for Next.js, session/JWT management
  bcryptjs        - Password hashing, industry standard
  Zustand         - Lightweight state, less boilerplate
